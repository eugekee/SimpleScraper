Система для автоматического сбора и анализа данных об акциях Сбербанка.

Проект предназначен для непрерывного мониторинга котировок акций Сбербанка с возможностью:
- Автоматического сбора данных с финансовых порталов
- Сохранения истории в Excel/CSV форматах
- Гибкой настройки интервалов мониторинга
- Настройка отображения графиков по данным в Excel/CSV
- Сбора подробной статистики работы системы

Установка и настройка

- Python 3.8+
- Chrome/Chromium браузер

pip install -r requirements.txt

Однократный запуск:

python main.py --single

Непрерывный мониторинг:

python main.py --continuous

Мониторинг с настраиваемым интервалом:

python main.py --continuous --interval 600
