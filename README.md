Система для автоматического сбора и анализа данных об акциях Сбербанка с помощью телеграмм бота.

Проект предназначен для непрерывного мониторинга котировок акций Сбербанка с возможностью:
- Автоматического сбора данных с финансовых порталов
- Сохранения истории в Excel/CSV форматах
- Гибкой настройки интервалов мониторинга
- Настройка отображения графиков по данным в Excel/CSV
- Сбора подробной статистики работы системы

Установка и настройка

Сам телеграмм Бот - @SberStockScraperbot

- Python 3.8+
- Chrome/Chromium браузер

Создайте в корневой папке .env с данными вашего API
TELEGRAM_BOT_TOKEN=ваш API

pip install -r requirements.txt

Доступные команды бота:

/start - начать работу с ботом

/help - показать справку по командам

/current - текущая цена акций Сбербанка

/stats - статистика по всем данным

/history - история цен за 7 дней

/history3 - история цен за 3 дня

/history1 - история цен за 1 день


Особенности реализации:

Прямая работа с Excel - бот читает данные напрямую из файла

Фильтрация данных - по дате и количеству записей

Ограничение вывода - показывается только 5 последних записей в истории

Статистика - подсчет средней цены, минимума, максимума

Обработка ошибок с информативными сообщениями

Логирование всех операций
